# BIEL STORE - Marketplace de Jogos

A **BIEL STORE** é uma plataforma de marketplace especializada na negociação segura de itens de jogos, contas, skins, moedas virtuais (diamantes, robux) e gift cards. Inspirada no modelo de segurança de intermediação (escrow) de plataformas consolidadas, ela garante segurança total para compradores e vendedores.

## 🚀 Tecnologias Utilizadas

- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Banco de Dados**: [PostgreSQL](https://www.postgresql.org/)
- **ORM**: [Prisma](https://www.prisma.io/)
- **Autenticação**: [NextAuth.js](https://next-auth.js.org/) ou [Clerk](https://clerk.com/)
- **Pagamentos**: Mercado Pago / Stripe (Gateway com suporte a Webhooks)
